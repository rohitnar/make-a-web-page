Definitions

WWW: World wide web
HTML: Hyper Text Markup Language

How it all ties together

You (through your browser) access the internet which in turn uses the HTTP (protocol) to access the various servers to access the information in the servers

HMTL is made up of

HTML Mark Up

B is used for Bold
Em is used for italics

HTML Attributes

"a" is used for Anchor which links the specific words to another webpage. For instance below
"img" is used for Images

White Space

All spaces in HTML are white spaces so unless specific commands are used, it will be presumed that everything is written in a flow.
"br" is line break
"p" is for one paragraph
The difference between the two is that while one is inline and other is a block.The block clubs all the words inside it as one cluster

Lesson 2: Creating a structured document with HTML

Developer tools (in a browser)

When writing code, programmers use special text editors (like sublime text for instance). These editors make the programmer's life easier. For example, text editors will automatically generate a closing HTML tag with an opening tag.

Lesson 3: Adding CSS style to HTML structure

What is CSS

CSS stands for Cascading Style Sheets and they give programmers a way to control the style of related HTML elements. This is done by giving similar HTML elements the same class name and then specifying the style that should apply to that class.

How to use CSS

For big projects its best to use CSS to avoid repetitions and thereby programming errors. To use CSS you have to link your html file to a seperate CSS file as per steps below
1. Write all your structured HTML in one file
2. In a seperate file, put all your CSS code
3. Add a head element to the top of the HTML file and add a link tag inside the head element to link the CSS file to the HMTL file.
